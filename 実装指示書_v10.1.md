## タスク 1-7: pg-boss セットアップ

```
設計書参照: Part F.1（エージェント実行フロー）、Part B.2（ジョブキュー）
優先度: 最高
```

`lib/queue/boss.ts` を作成してください:

```typescript
import PgBoss from 'pg-boss'

let boss: PgBoss | null = null

export async function getBoss(): Promise<PgBoss> {
  if (boss) return boss
  boss = new PgBoss({
    connectionString: process.env.SUPABASE_DB_URL,
    // pg-boss v10 の設定
    schema:           'pgboss',
    monitorStateIntervalSeconds: 30,
    archiveCompletedAfterSeconds: 86400,  // 24時間後にアーカイブ
  })
  await boss.start()
  return boss
}

export async function stopBoss() {
  if (boss) await boss.stop()
}
```

ワーカー起動スクリプト `apps/web/src/workers/start.ts` を作成:
- PropertyIntakeAgent のワーカー登録
- ImprovementAgent のワーカー登録
- OrchestrationLoop の cron 登録（5分間隔）
